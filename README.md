# Reoria
 Reoria is an online adventure role playing game made with Unity.

# Downloading Assets
The assets for the game are only available to deveoplers. If you are a developer and need access to these please contact XerShade.
After you have been granted access to the assets repository you need to run this command in the assets folder: `git clone "https://github.com/XerShade/Reoria.Resources" Resources`
